# Milestone v1.0: Codebase Context

**Status:** SHIPPED 2026-01-11

## Overview

Core codebase context features for AI-powered diagram generation. Users can drag & drop files/folders or paste GitHub URLs to provide code context for AI-assisted diagram creation.

## Milestone Summary

**Key Decisions:**

- Drag & drop as primary interaction (matches user expectation)
- GitHub public repos only (avoids auth complexity)
- AI picks diagram type (reduces friction, leverages AI intelligence)
- Context stored per-conversation (simple, no persistence needed)
